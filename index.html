What is a web server?

A web server is a computer system that processes requests via HTTP. It can refer to the entire system or to the software that handles HTTP requests.

The primary function of a web server is to store, process and deliver web pages to clients.

For example, if you type in a web address on your browser, it will send a HTTP-request for that specific website. The web server will then accept the request and replies with a HTTP-response.

A HTTP-response usually contain a HTML document, which may include images, style sheets and scripts in addition to text content.
